# 🩸 Multimodal Blood Report Analyzer

A multimodal AI application that analyzes blood report images using Google's Gemini Vision model. The system processes uploaded blood report images, extracts important medical information, and generates AI-powered health insights.

---

## 🚀 Features

- 📄 Analyze blood report images
- 👁️ Image understanding using Gemini Vision
- 🩺 Extract blood test values
- 🤖 AI-generated medical insights
- 📋 Summarize laboratory findings
- ⚡ Fast multimodal analysis

---

## 🛠️ Tech Stack

- Python
- Google Gemini Vision API
- Pillow (PIL)
- Python Dotenv
- Jupyter Notebook

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/codinghub37/multimodal-blood-report-analyzer.git
```

### Navigate to the Project

```bash
cd multimodal-blood-report-analyzer
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file and add your Gemini API key.

```env
GOOGLE_API_KEY=your_api_key_here
```

---

## ▶️ Run the Project

```bash
jupyter notebook
```

Open `multimodel_demo.ipynb` and run all notebook cells.

---

## 📚 Workflow

1. Load the blood report image.
2. Send the image to the Gemini Vision model.
3. Extract medical information from the image.
4. Analyze blood test parameters.
5. Generate AI-powered health insights.
